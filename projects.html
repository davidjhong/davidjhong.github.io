Cutie Calculator

I created a visual equation solver akin to Photomath using Python and OpenCV. The program classifies basic math operators through gray pattern and number extraction. Key frameworks I utilized include OpenCV for image processing, JSON for data storage, OS commands, and Tkinter for the GUI. This project allowed me to gain experience with computer vision techniques like optical character recognition. Overall, it demonstrates my Python skills and ability to develop an end-to-end application from image processing to user interface.

Rose Lock

I developed a 3 layer home security system using facial recognition and AI through a custom website. The system connects an Arduino board to a password-protected website I built with HTML, CSS, and ReactJS. My C++ code integrates the hardware with the website and Python functions for remote monitoring. Through this project I improved technical skills including programming, web development, and hardware integration, cutting the time required to manage home security by 50%. Overall, this project showcases my ability to create an end-to-end security system with AI and web connectivity.
